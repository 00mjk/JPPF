Java Parallel Processing Framework

Third-Party Libraries
JFreeChart 1.0.3: A great charts library used in the JPPF admin console. You will need the files jfreechart-1.0.3.jar and jcommon-1.0.6.jar in the JPPF/lib/JFreeChart folder.
Substance Look & Feel 3.1: The awesome look and feel used in all JPPF swing-based UIs. JPPF requires subtance-3.1.jar (i.e. rename the substance.jar file) in the JPPF/lib/Substance folder.
Apache log4j 1.2.9: The famous logging library from the Apache Software Foundation You will need the file log4j-1.2.9.jar in the JPPF/lib/log4j folder.
Rhino 1.6R2: A great library for integration of Javascript and Java. JPPF uses it in the admin console, through Javascript fragments embedded in the XML-defined UI components. You will need the js.jar file in the JPPF/lib/Rhino folder. May not be needed if you are using JDK 1.6 (untested).
Saverbeans SDK 0.2: This is the API and tools used to generate and deploy JPPF nodes as screensavers (JPPF@Home). Extract the saverbeans-sdk-0.2.zip file as the JPPF/lib/saverbeans-sdk folder.
Apache Commons IO 1.2: Used by the JPPF nodes for common IO operations using URLs and streams (i.e. HTTP and FTP based IO). JPPF requires the commons-io-1.2.jar file in the JPPF/lib/ApacheCommons folder.
UJAC-UI 0.9.6: This is a library for the JEdit editor, used by the JPPF GUI Builder, which is still in a very early and experimental stage. JPPF needs the ujac-ui.jar file in the JPPF/lib/UJAC-UI folder.
