Java Parallel Processing Framework on
Description The Java Parallel Processing Framework is a suite of software libraries and tools providing convenient ways to parallelize CPU-intensive processing.
It is written in the Java programming language and is platform-independant.

Major features include:
  • an API to delegate the processing of parallelized tasks to local and remote execution services
  • a set of APIs and user interface tools to administer and monitor execution services
  • real-time adaptive load balancing capabilities
  • scalability up to an arbitrary number of processing nodes
  • limited intrusiveness for existing or legacy code
  • fully documented APIs, administration guide and developer guide
Current status Version 0.0.1 - pre-alpha
