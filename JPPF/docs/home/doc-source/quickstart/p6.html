6 Links and pointers

We hope that this quick start guide has provided you with an understanding of how you can use JPPF, as well as insights as to how you can integrate grid computing into your development plans.

If you wish to go further, we recommend a number of resources that will help you find the information you need:

  • To learn about the latest JPPF news and releases, follow this link to the JPPF home page
  • The JPPF documentation has detailed information on JPPF architecture, usage, programming and features
  • The forums will provide you with the possiblity to find help and support, report issues, as well as share your experience with other users and the JPPF developers
  • If you wish to know what we're working on, or submit ideas to improve JPPF, we invite you to look at, and contribute to our feature requests
  • If you want to know about current and past known issues, do not hesitate to browse our bugs database
  • If you want to learn more about grid computing, you can start with our links page
