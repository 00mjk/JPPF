5 Running the client

This is the final stage of this tutorial. We will see how to write a script to run our "Hello World" application.

5.1 Script requirements

What we need to do in our script is as follows:

  • Define the command to launch Java
  • Specify the classpath, including:
    • the classes folder, where the compiled classes of this tutorial's source code are located
    • the JPPF libraries: lib/jppf-client.jar, lib/jppf-common.jar, lib/jppf-common-node.jar
    • the Apache commons-logging library: lib/commons-logging-1.1.jar
    • the config folder, which contains the JPPF client and logging configuration files: jppf-client.properties, commons-logging.properties, logging.properties
  • Specify the Java system property for the location of the JPPF configuration file: -Djppf.config=jppf-client.properties
  • Specify the Java system property for the location of the logging configuration file: -Djava.util.logging.config.file=logging.properties
  • Specify the main class of our application: org.jppf.helloworld.HelloWorldRunner

5.2 Windows script

$template{name="code_start"}$ set JAVA_CMD=%JAVA_HOME%\bin\java set CP=classes set CP=%CP%;lib\jppf-client.jar set CP=%CP%;lib\jppf-common.jar set CP=%CP%;lib\jppf-common-node.jar set CP=%CP%;lib\commons-logging-1.1.jar set CP=%CP%;config set SYS_PROPS=-Djppf.config=jppf-client.properties set SYS_PROPS=%SYS_PROPS% -Djava.util.logging.config.file=config/logging.properties call %JAVA_CMD% -cp %CP% %SYS_PROPS% org.jppf.helloworld.HelloWorldRunner$template{name="code_end"}$

This script can be downloaded here: runHelloWorld.cmd

5.3 Unix/Linux script

$template{name="code_start"}$ #!/bin/sh JAVA_CMD=$JAVA_HOME/bin/java CP=./classes CP=$CP:./lib/jppf-client.jar CP=$CP:./lib/jppf-common.jar CP=$CP:./lib/jppf-common-node.jar CP=$CP:./lib/commons-logging-1.1.jar CP=$CP:./config SYS_PROPS=-Djppf.config=jppf-client.properties SYS_PROPS=$SYS_PROPS -Djava.util.logging.config.file=config/logging.properties $JAVA_CMD -cp $CP $SYS_PROPS org.jppf.helloworld.HelloWorldRunner $template{name="code_end"}$

This script can be downloaded here: runHelloWorld.sh

5.4 Executing remotely

To enable remote execution of the tasks, you will have to download and install the JPPF driver and node packages. To do so, please follow these instructions:

  • Instructions to download, install and run a JPPF driver
  • Instructions to download, install and run a JPPF node

Please note that you will need to have Apache Ant installed on your machine, prior to running the driver or node.

Once you have a a driver and node running, you may disable the local execution in the client configuration.
To do so, open the ./config/jppf-client.properties file in a text editor and replace the following line:

$template{name="code_start"}$jppf.local.execution.enabled = true$template{name="code_end"}$

with:

$template{name="code_start"}$jppf.local.execution.enabled = false$template{name="code_end"}$
