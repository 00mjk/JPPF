1 Introduction

The goal of this quick start tutorial is to show you how to write a full JPPF application and execute it in your environment. It is delivered as a full self-contained application with its own scripts for building and running it.

1.1 What you will learn

  • An overview of the JPPF capabilities
  • How to write a fully functional JPPF application
  • How to integrate your existing code with JPPF
  • How to run a sample JPPF application
  • How to find further information and support for advanced use of JPPF

1.2 Organization of this guide

The code in this tutorial can be viewed, built and executed in 3 ways:

  • From the command line: Windows and Linux scripts are included for executing the application. To build from the source, you will need to use one of the other 2 ways.
  • As an Apache Ant project, using the provided build.xml build file. The targets are "build" for building the code, and "run.helloworld" to execute the application.
  • As an Eclipse project: simply import this Quick Start Guide as an Eclipse project (Eclipse 3.4 or later required), and use the provided launch configuration to run it.

1.3 Prerequisites

To effectively run through this tutorial, you should have the following:
  • Java 1.5 or later installed in your environment
  • Apache Ant installed in your environment
  • The JPPF package containing the present tutorial
  • The JPPF driver and node binary packages, which you can get from the JPPF download page
