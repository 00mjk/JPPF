System Requirements

  • Any platform with Java 2, Standard Edition (J2SE) 5.0 or later.
  • Apache Ant 1.6.2 or later, for building and running the application and examples.

Installing

Framework distribution

Simply unzip the jppf-full-x.y.z-nnnn.zip file in a location of your choice. This gives you the JPPF framework, including the sources, demos, and documentation.

Runtime distribution

Unzip the jppf-full-x.y.z-nnnn-runtime.zip file in a location where you intend to run a node or the UI from. The runtime contains the minimum set of required files and libraries for running a node and the GUI. You'll have to change the configuration properties in the config folder as described in the user guide.

Running the Demo

  1. Go to the jppf-x.y.z/examples/Demo folder
  2. type ant run.local to run the example
This script launches 3 separate processes and will take 100% CPU on single-CPU machines.
Please give it some time, the demo may take 30 seconds or more to execute.

Running the monitoring UI

  1. Go to the jppf-x.y.z/examples/Demo folder
  2. type ant run.ui to run the Swing UI

More information can be found in the FAQ

Description of the distribution

After unzipping, you will see the following folder structure, under the JPPF root folder:
  • bin contains the scripts and config files for building the JPPF distribution, including the zip file you downloaded, the javadoc and the demo
  • classes contains the compiled classes of JPPF, excluding the demo
  • docs has two subfolders: api for the JPPF api documentation, and manual for a quick guide and tutorial to help you use JPPF through a documented demo that is part of the distribution
  • examples is the root container for all demos and examples distributed with the framework
  • lib contains all thrid party libraries required to run JPPF nodes and the JPPF GUI. The following third party libraries are required:
    • log4j: the well known Apache logging framework
  • src contains all sources, fully documented, of the framework
